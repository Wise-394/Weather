# 🌤️ Weather App (The Odin Project)
This is a weather application built as part of __The Odin Project__ curriculum. It focuses on working with external APIs, asynchronous JavaScript (Promises & async/await), and dynamic DOM manipulation to fetch and display real-time weather data.

## 🚀 Features

* Search weather by city name
* Displays current weather condition
* Toggle between **Celsius** and **Fahrenheit**
* Loading state with async feedback while fetching data
* Error handling for invalid city names or failed requests

## 🧠 Key Concepts Practiced

* **API Integration** — Fetches live data from the [WeatherAPI](https://www.weatherapi.com/) (or OpenWeatherMap)
* **Async/Await** — Handles asynchronous requests using modern ES6+ syntax
* **Promises & Fetch API** — Makes HTTP GET requests and processes JSON responses
* **Error Handling** — Catches network failures and invalid inputs gracefully
* **DOM Manipulation** — Dynamically updates the UI based on API response data

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6+) — async/await, Fetch API, Promises
